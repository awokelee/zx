# 项目目录

> 项目目录介绍

```js
├─build                 // 构建
├─config                // 配置
├─node_modules          // 配置
├─src                   // 源文件
│  ├─assets             // 资源(css、图片)
│  ├─components         // 公用组件
│  └─router             // 路由
├─static                // 静态资源
│
├─.babelrc              // js新语法转换(向下兼容)
├─.editorconfig         // 编辑器配置
├─.gitignore            // git忽略
├─.postcssrc.js         // postcss(添加各浏览器兼容前缀)
├─index.html
├─package.json          // 包配置文件(类似pom.xml)
└─README.md             // 项目说明文档(一般怎么运行项目都在里面)
```
