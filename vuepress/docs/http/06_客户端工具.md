# 客户端工具

### 浏览器

### curl

`Gitbash` 自带.

`curl baidu.com` 返回百度内容, 从下面可以看出 就三行代码. `refresh` 重定向到 `http://www.baidu.com/`

```bash
[root ~]# curl baidu.com
<html>
<meta http-equiv="refresh" content="0;url=http://www.baidu.com/">
</html>
```
