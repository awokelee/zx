# 图片命名

```text
1、导航命名：menu.gif 
如：menubg .gif（导航的背景图）

2、会员登录：login.gif 
如：loginbg.gif （会员登陆的背景图）
3、搜索命名：search.gif 如：search_bg.gif （搜索的背景图）

4、小 图 标：ico_数字.gif 如：ico_001.gif

5、线的命名：line_X_颜色.gif 
如：line_X_red.gif（红色横向虚线）
说明：line只命名虚线line_Y_red.gif（红色纵向虚线）

6、广告命名：ad_数字.gif 如：ad001.gif

7、其它栏目的图片：以栏目名的第一个字母.gif如：xwzx_bg.gif （新闻中心背景） cpzx_l.gif （产品中心的左边图）

8、产品与栏目热点图片： pic_数字.gif 如：pic_001.gif说明：上、下、左、右 可以缩写为T、B、L、R
```
