# 图片的命名规范及脚本文件和动态文件命名规范

```text
1   图片的名称分为头尾两部分，用下划线隔开，头部分表示此图片的大类性质，例如：广告、标志、菜单、按钮等等

2   常规命名

2.1   放置在页面顶部的广告、装饰图案等长方形的图片取名：banner

2.2   标志性的图片取名为：logo

2.3   在页面位置不固定并且带有链接的小图片我们取名：button

2.4   在页面上某一个位置连续出现，性质相同的链接栏目的图片我们取名：menu

2.5   装饰用的照片我们取名：pic

2.6   不带链接表示标题的图片我们取名：title

3   范例

   banner_sohu.gif

   banner_sina.gif

   menu_aboutus.gif

   menu_job.gif

   title_news.gif

   logo_police.gif

  logo_national.gif

  pic_people.jpg

二   脚本文件和动态文件命名规范

1   一般用脚本功能的英语小写缩写命名

1.1   实际模块

   例如：广告条的javascript文件名为ad.js弹出窗口的javascript文件名为pop.js

1.2   共用模块

   JS文件命名：英文命名，后缀.js.共用common.js.basic.js

1.3   外部资源

  Jquery.min.js

   Jquery.validate.js

   Jquery.date.js

2   动态语言文件命名原则：见名知意

2.1   以性质描述，描述可以有多个单词，用"_"隔开，性质一般是该页面的概要

  范例：register_form.aspx

   register_post.aspx

   topic_lock.aspx

2.2   不同模块之间，可以使用不同的前缀来进行区分
```
