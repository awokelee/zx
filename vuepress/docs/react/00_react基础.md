# react 介绍

[React JS 官网](https://reactjs.org/)

`Facebook` 2013 年 5 月开源, `React 16` 版本前叫做 `React JS`, 之后叫做 `React Fiber` , 因为在 **底层事件循环** 中加入了 **优先级** 等概念, 可以利用事件循环的碎片时间执行 **高优先级** 的用户交互, 提升用户体验.

`React` 只支持 `IE 8` 及以上, 如果要兼容更低版本, 则不能使用 `React`.
